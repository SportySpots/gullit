#dev server
To avoid CORS issues, run the proxy server with `yarn proxy`. Then start the dev server:
 
    yarn dev
    
#storybook
    
    yarn storybook
    
#run tests

    yarn test
    
#build

    yarn build
    
#production server

    yarn build && yarn start
